# <Badge function /> classnames()

The classnames() method returns the class names conditionally joining.

## Usage

```ts copy
import { classnames } from '@ponsetya/javascript'
```

## Syntax

```ts
classnames(...args: ClassNameArgument[]): string
```

## Parameters

<dl>
  <dt>**...args:** `ClassNameArgument[]`</dt>
  <dd>Class name references.</dd>
  <dt>**Returns:** `string`</dt>
  <dd>A String representing the name of the generated class.</dd>
</dl>

## Examples

```ts copy
console.log(classnames('foo', 'bar')) // 'foo bar'
console.log(classnames({ foo: true, bar: false })) // 'foo'
console.log(classnames({ foo: true, bar: true })) // 'foo bar'
console.log(classnames({ foo: 'baz' }, { bar: true })) // 'foo-baz bar'
```

## Type Aliases

### ClassNameValue

```ts
type ClassNameValue = string | number | boolean | undefined | null
```

### ClassNameValueObject

```ts
type ClassNameValueObject = Record<string, ClassNameValue>
```

### ClassNameArgument

```ts
type ClassNameArgument = ClassNameValue | ClassNameValueObject
```
