# Getting Started

## Installation

    ```bash copy

    ```

  </Tab>
  <Tab>
    For using the pnpm package manager.

    ```bash copy
    pnpm install @ponsetya/javascript
    ```

  </Tab>
  <Tab>
    For using the npm package manager.

    ```bash copy
    npm install @ponsetya/javascript
    ```

  </Tab>
</Tabs>

## Usage

By default you can include all api and functions.

```ts copy
import javascript from '@ponsetya/javascript'

console.log(javascript.uid()) // 25-character unique string
```

If you want to use it as a module, you need to import the module.

```ts copy
import { uid } from '@ponsetya/javascript'

console.log(uid()) // 25-character unique string
```
