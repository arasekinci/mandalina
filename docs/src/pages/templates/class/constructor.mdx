# Rectangle() constructor

The `Rectangle()` constructor creates [Rectangle](/templates/rectangle) objects.

## Syntax

```ts
new Rectangle(width)
new Rectangle(width, height)
```

### Parameters

<ParameterList>
  <Parameter name="width" description="A string representing the name of the width." type="number" />
  <Parameter name="height" description="A string representing the name of the height." type="number" optional />
</ParameterList>

## Examples

```ts copy
const rectangle = new Rectangle(200, 100) // [Rectangle](/templates/rectangle)
```
