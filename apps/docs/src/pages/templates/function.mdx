## Types

### Type aliases

#### AccountData

```ts
/**
 * AccountData is a TypeScript type representing additional data associated with an account.
 */
type AccountData = {
  /** An array of strings representing the account holder's hobbies. */
  hobbies: string[]

  /** Enumeration representing the type of account. */
  type: '[AccountType](#accounttype)'

  /** An object representing reading and writing permissions. */
  permits: {
    /** A boolean indicating whether the account has reading permission. */
    read: boolean

    /** A boolean indicating whether the account has writing permission. */
    write: boolean
  } | null
}
```

### Interfaces

#### Account

```ts
/**
 * Account is an interface representing the structure of an account.
 *
 * @template T - A parameter representing the dynamic T type.
 */
interface Account<T> {
  /** A string representing name. */
  name: string

  /** A string representing age. */
  age: string

  /** An optional property representing additional data associated with the account. */
  data?: '[AccountData](#accountdata)'

  /** A generic property `T` representing user-specific settings associated with the account. */
  settings?: T
}
```
